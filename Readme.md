# 📝 Django Todo App

A simple and efficient **Task Management Web Application** built with **Django** that allows users to create, manage, and organize their daily tasks.

🚀 The application is deployed online and accessible from anywhere.

---

## 🌐 Live Demo

🔗 https://django-todo-app-5.onrender.com

## ✨ Features

* ➕ Add new tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 📋 View all tasks in a clean interface
* ⚡ Fast and lightweight Django backend

## ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Tharun-301/django-todo-app.git
cd django-todo-app
```

### 2️⃣ Create virtual environment (optional but recommended)

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Apply database migrations

```bash
python manage.py migrate
```

### 5️⃣ Run the development server

```bash
python manage.py runserver
```

Open in browser:

```
http://127.0.0.1:8000
```

---

## 🔑 Admin Panel (Optional)

Create an admin user:

```bash
python manage.py createsuperuser
```

Access the admin dashboard:

```
/admin
```
